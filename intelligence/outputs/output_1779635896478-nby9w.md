class Node {
    int data;
    Node next;

    Node(int data) {
        this.data = data;
        this.next = null;
    }
}

class CircularLinkedList {
    Node head;

    CircularLinkedList() {
        this.head = null;
    }

    void insert(int data) {
        Node newNode = new Node(data);
        if (head == null) {
            head = newNode;
            newNode.next = head;
        } else {
            Node temp = head;
            while (temp.next != head) {
                temp = temp.next;
            }
            temp.next = newNode;
            newNode.next = head;
        }
    }

    void delete(int key) {
        if (head == null) {
            return;
        }

        Node current = head;
        Node prev = null;

        do {
            prev = current;
            current = current.next;
        } while (current != head && current.data != key);

        if (current == head) {
            if (current.next == head) {
                head = null;
            } else {
                Node temp = head;
                while (temp.next != head) {
                    temp = temp.next;
                }
                temp.next = current.next;
                head = current.next;
            }
        } else if (current != null) {
            prev.next = current.next;
        }
    }

    void display() {
        if (head == null) {
            System.out.println("List is empty");
            return;
        }

        Node temp = head;
        do {
            System.out.print(temp.data + " ");
            temp = temp.next;
        } while (temp != head);
        System.out.println();
    }
}

# Explanation

 This Java code defines a Circular Linked List data structure with insertion, deletion, and display functionalities.

1. **Node Class**: Represents a single node in the circular linked list. Each node contains an integer data and a reference to the next node in the list (or itself in a circular list).

```java
class Node {
    int data;
    Node next;

    Node(int data) {
        this.data = data;
        this.next = null;
    }
}
```

2. **CircularLinkedList Class**: Represents the circular linked list itself. It contains a head node that points to the first node in the list (or null if the list is empty).

```java
class CircularLinkedList {
    Node head;

    CircularLinkedList() {
        this.head = null;
    }
```

3. **insert(int data)**: Adds a new node with the given data to the end of the circular linked list. If the list is empty, the new node becomes the head.

```java
void insert(int data) {

    }
}
```

4. **delete(int key)**: Removes the first node containing the given key from the circular linked list. If the list is empty, no action is taken. If the key is not found, the list remains unchanged.
