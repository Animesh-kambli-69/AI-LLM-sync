class Node {
    int data;
    Node next;

    Node(int data) {
        this.data = data;
        this.next = null;
    }
}

class CircularLinkedList {
    Node head;

    void insert(int data) {
        Node newNode = new Node(data);
        if (head == null) {
            head = newNode;
            newNode.next = head;
        } else {
            Node temp = head;
            while (temp.next != head) {
                temp = temp.next;
            }
            temp.next = newNode;
            newNode.next = head;
        }
    }

    void delete(int key) {
        if (head == null) {
            return;
        }

        Node current = head;
        Node prev = null;

        do {
            prev = current;
            current = current.next;
        } while (current != head && current.data != key);

        if (current == head) {
            if (current.next == head) {
                head = null;
            } else {
                Node temp = head;
                while (temp.next != head) {
                    temp = temp.next;
                }
                temp.next = head.next;
                head = head.next;
            }
        } else if (current != null) {
            prev.next = current.next;
        }
    }

    void display() {
        if (head == null) {
            System.out.println("List is empty");
            return;
        }

        Node temp = head;
        do {
            System.out.print(temp.data + " ");
            temp = temp.next;
        } while (temp != head);
        System.out.println();
    }
}

public class Main {
    public static void main(String[] args) {
        CircularLinkedList list = new CircularLinkedList();
        list.insert(10);
        list.insert(20);
        list.insert(30);
        list.insert(40);

        list.display();

        list.delete(20);
        list.display();

        list.delete(10);
        list.display();

        list.delete(40);
        list.display();

        list.delete(30);
        list.display();
    }
}

# Explanation

 This Java code defines a Circular Linked List data structure and provides methods for inserting, deleting, and displaying elements in the list.

1. **Class Node**: Represents a single node in the circular linked list. Each node has an integer data field and a reference to the next node in the list, which can point to either the head or the tail of the list, forming a circular structure.

2. **Class CircularLinkedList**: Represents the circular linked list itself. It has a reference to the head node.

   - **insert(int data)**: Adds a new node with the given data to the end of the circular linked list. If the list is empty, the new node becomes the head and tail.

   - **delete(int key)**: Removes the first node with the given key from the circular linked list. If the key is not found, the method does nothing. If the node to be deleted is the head, the head is updated accordingly.

   - **display()**: Prints the elements of the circular linked list in a circular fashion, i.e., starting from the head and continuing until the next node is the head again.

3. **Class Main**: Contains the main method that creates an instance of the CircularLinkedList, inserts some elements, deletes some elements, and displays the list at various stages.

The key components of this code are the Node and CircularLinkedList classes, which define the structure and behavior of the circular linked list. The use case of this code is to provide an implementation of a circular linked list data structure in Java, which can be useful in various algorithms and data structures problems that require cyclic data structures.
